❤️ Heart Disease Analysis Dashboard
📌 Project Overview: 
This project presents an interactive Heart Disease Analysis Dashboard that provides insights into heart disease trends based on gender and age groups. The dashboard is designed to help users easily understand patterns, risk distribution, and key factors influencing heart disease.

🎯 Objectives
Analyze heart disease distribution across different genders
Identify high-risk age groups
Provide clear and interactive visual insights
Support data-driven decision making

📊 Features
Gender-wise comparison of heart disease cases
Age group segmentation analysis
Interactive visualizations for better understanding
Clean and user-friendly dashboard design

🛠️ Tools & Technologies Used
Power BI (for dashboard creation)
Excel / CSV (for dataset handling)
Data Visualization techniques

📂 Project Structure
├── dataset/
│   └── heart_disease_data.csv
├── dashboard/
│   └── heart_disease_dashboard.pbix
├── images/
│   └── dashboard_preview.png
└── README.md

📈 Key Insights
Certain age groups show higher susceptibility to heart disease
Gender-based differences are visible in disease occurrence
Patterns help in identifying potential risk factors

🚀 How to Use
Download the .pbix file from the repository
Open it using Power BI Desktop
Interact with filters and visuals to explore insights
